# Profiles

This folder stores per-model characterization generated by the local import
workflow.

Each model profile should include:

- static characteristics from the GGUF metadata
- inferred datatype and local file footprint
- estimated KV-cache growth by context size
- live context stress results against the real ROCm runtime
- a short markdown summary for future supervisors

Use the CLI to populate a model profile:

```bash
.venv/bin/python -m llm_gallery.cli import-model --model <path-to-model.gguf>
```
